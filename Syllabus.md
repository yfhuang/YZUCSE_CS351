# CS351/IN321: AI-assisted Software Development - Syllabus

## Policies
- **Attendance:** On-site attendance is required.  
- **Academic Integrity:** Plagiarism and misuse of AI tools are strictly prohibited.  
- **Late Submission:** Late submissions will not be accepted.  

## Prerequisites
- Proficiency in programming with modern languages (e.g., Python, JavaScript, or C++)  
- Familiarity with version control systems such as [Git](https://git-scm.com/)  
- Basic understanding of AI concepts and tools  

## Learning Outcomes
Upon successful completion of this course, students will be able to:  
1. Understand the role of AI in software development.  
2. Utilize AI tools to enhance coding, debugging, and testing.  
3. Evaluate the ethical implications of AI in software engineering.  
4. Develop software projects incorporating AI-assisted techniques.  

## Topics Covered
1. Introduction to AI in Software Development  
2. AI-Assisted Coding Tools  
3. AI for Debugging and Testing  
4. AI in Code Review and Refactoring  
5. Ethical Considerations in AI-Assisted Development  
6. Case Studies and Applications
